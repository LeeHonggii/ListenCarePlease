### 📋 다이어그램 해설

1. **INPUT:** 파일이 들어갑니다.
2. **LOADER:** 랭체인의 도구들이 파일 포장지를 뜯고 알맹이(텍스트)만 꺼냅니다.
3. **CHAIN (체인):** 여기가 핵심입니다.
    - **Prompt:** 텍스트에 "이거 분석해"라는 명령어를 붙입니다.
    - **LLM:** AI가 명령대로 분석 결과를 텍스트로 뱉습니다.
    - **Parser:** AI의 말을 파이썬이 알아듣는 데이터(JSON)로 깔끔하게 바꿉니다.
4. **OUTPUT:** 최종적으로 우리가 코드에서 쓸 수 있는 리스트나 딕셔너리가 나옵니다.

## Output Paser
[프롬프트] --> 🤖 LLM --> "네! 여기요: ```json {...}```" (문자열)
                                      ⬇️
                            ⚙️ Output Parser (자동 변환기)
                                      ⬇️
                                 {데이터} (완성)

chain = prompt_template | llm | parser

## 발생 문제
- LangChain 변경 후, docx file error
- pdf, docx 투입 간 오류 생성
    - install pypdf docx2txt 추가
- 작일 문제점 지속 발생
    - 해결 완료
    - prompt 강화, 함수 추가
