# 파이프라인
전사 텍스트 입력
→ 문장 및 문맥 파악
→ 단어 선정
→ 각 사전 조회
→ LLM 판단 및 요약
    → 사전에 없는 단어는 LLM이 뜻 생성
→ 결과 dictionary 출력
    → {단어 : 뜻}

# 현재 진행 과정
임의의 단어 list 입력([단어, 단어, 단어])
→ 각 사전 API로 뜻 파악
→ 파악된 뜻 종합 후 LLM 입력
→ LLM 요약 및 공란 뜻 추가
→ 테스트 출력

# LLM 입력 결과
1차 결과 - 단순 단어 리스트 제공으로 뜻 출력 테스트
vocab_list = ["사람", "기억", "상상", "돈다발"]

📘 최종 vocab 사전 결과:
사람 : 사람은 생각하고 언어를 사용하며 도구를 만들어 쓰고 사회를 이루어 사는 동물로, 지구에 사는 인간 전체(인류)를 가리킵니다.
기억 : 기억은 경험하거나 배운 것을 뇌에 저장해 나중에 다시 떠올리는 능력이다. 이렇게 저장된 정보는 시간이 지나도 남아 앞으로의 행동에 영향을 준다.
상상 : 상상(상상력)은 경험하지 않았거나 현재 보이지 않는 것을 머릿속으로 그려보는 능력이다. 보이지 않는 장면이나 감각, 개념을 정신적 이미지로 만들어내는 마음의 힘을 뜻한다.
돈다발 : 지폐 여러 장을 한데 묶어 둔 것, 즉 현금 뭉치를 뜻합니다.

1-2차 결과 : 전문 용어라 판단되는 단어들로 테스트
vocab_list = ["LLM", "STFT", "Fine-tuning", "kernel"]

📘 최종 vocab 사전 결과:
LLM : LLM(대형 언어 모델)은 수십억 개의 매개변수를 가진 인공신경망으로, 방대한 양의 라벨 없는 텍스트를 자기지도 학습으로 학습해 사람처럼 텍스트를 이해하고 생성하는 모델입니다.
STFT : STFT(Short-Time Fourier Transform)는 신호를 짧은 시간 구간으로 나눈 뒤 각 구간에 푸리에 변환을 적용해 주파수 성분이 시간에 따라 어떻게 변하는지 보여주는 분석 기법입니다. 음성·음향이나 진동 신호 등 시간에 따라 변하는 주파수 특성을 살필 때 자주 사용됩니다.
Fine-tuning : Fine-tuning(파인튜닝)은 어떤 시스템이나 계획을 더 잘 작동하게 하기 위해 작은 부분들을 세밀하게 조정하여 성능을 최적화하는 것을 뜻합니다.
kernel : kernel은 어떤 것의 핵심이나 중심이 되는 부분을 뜻하는 말입니다. 예를 들어 과일의 씨앗이나 컴퓨터에서 운영체제의 핵심 부분을 가리킬 때 쓰입니다.
