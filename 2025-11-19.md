1. 발화 점유율 (Share of Voice, SoV)
    
    $$
    SoV_k = \frac{\sum^{n}_{i=1}(End_{i,k}-Start_{i,k})}{Total Meeting Duration} * 100
    $$
    
2. 발화 빈도 및 턴 테이킹 (Turn-Taking Frequency)
    1. 분석 : 한번에 길게(3분이상?) 말하는 스타일인지, 짧게 자주(10초 내외) 치고 들어가는 스타일인지 구분합니다.
    2. 의미:
        1. 높은 빈도 + 짧은 길이 : 활발한 토론, 아이디에이션 (긍정)
        2. 낮은 빈도 + 긴 길이  : 일방적인 보고, 강의식 회의 (부정적, 지루함 유발 가능성)
    
    >>  **턴은 세그먼트 숫자로 생각하면 되겠다**
    
3. 발화 간 침묵 시간 (Slinence Gap Analysis)
    1. 의미 : 발화가 끝난 시점과 다음 발화가 시작되는 시점 사이의 간격을 분석
        1. 긴 침묵 : 아이디어가 고갈 or 분위기 경직
        2. 짧은 침묵 : 몰입도 높고 상호작용이 빠름

## 정보량 및 정보 엔트로피

- 정보량 : 발화된 문장이 얼마나 예측 불가능하고 새로운가 (듣고 깜놀하는 정도)
    - 뻔한 말 (”네”, “알겠습니다.”,”안녕하세요”)는 정보량이 0에 가까움
    - 새로운 아이디어, 전문용어, 구체적인 수치는 확률이 낮아 정보량이 높다.
- 엔트로피 : 회의 전체의 불확실성 또는 다양성의 평균
    - 엔트로피가 높다.  = 회의 내용이 다채롭고 예측 어렵다.

- 최대 정보량은 최고의 회의가 아니다.
    - 인지 부화 문제
    - 합의와 수렴의 부재
    - 결속력 저하
- 정보 엔트로피가 점차 낮아진다의 의미
    - 정보 이론적 관점  : ‘우리가 무엇을 할지 확실해지고 있다.’
    - 언어학적 관점 : 어휘의 동기화
    - 또한  감소하는 기울기 또한 중요하다.
        - 지속적 우하향 ⇒ 다양한 의견 후 합의 도달
        - 처음부터 끝까지 낮음 (→ 바닥) ⇒ 새로운 아이디어 없음, 상명하복
        - 처음부터 끝까지 높음 (→ 천장) ⇒ 결론 없이 끝없는 노쟁
        - 급격한 하락 ⇒ 충분한 논의 없이 성급한 결론
- 방식
    - 정보 엔트로피는 슬라이딩 윈도우 방식
    - 정보량은 문장마다의 정보량 분석 ⇒  심전도 모양이 나올
        - 회의 요약 에 활용 가능하다. → 정보량이 높은 문장이 핵심 문장
        - 회의 밀도 파악
            - 중간중간 높은 스파이크 + 이를 소화하는 낮은 정보 → 좋음
            - 스파이크 없이 바닥을 긴다. → 지루함
            - 스파이크가 쉴 새 없이 튄다. → 어려움
- 주의 : 긴문장은 당연히 정보량이 많다. → 정규화 필요하다.

# 생각해보면 좋을 지표

- perplexity 변화 추이 : 우하향 하는 모습
    - 정보 엔트로피랑 매우 유사하다. 앞의 문장을 보고 현재 문장이 얼마나 갑작스러운 문장인지를 평가한다.
    - 
- 어휘 다양성 (TTR) : (고유 단어 수 / 전체 단어 수)
    
    높다 ⇒ 주제가 산만하다.
    
    낮다 ⇒ 앵무새
    
    **그럼 얼마가 적당한거지?**
    
- 상호 정보량 (Pointwise Mutual Information, PMI) : 대화의 내용이 얼마나 서로 연결되어있나
    - 주제 응집성 평가 : 키워드 추출 후 쌍별로 PMI 계산
        - 높은 PMI : 키워드들이 서로 긴밀하게 연관
        - 낮은 PMI : 뜬금없는 단어들이 나열됨
    - 질의응답 적합성 평가  : QA 쌍에 대한 PMI 계산
        - 높음 : 관련 단어를 사용하여 답변
        - 낮음 : 전혀 상관없는 단어로 답변
    - 화자간 케미 분석 : A B간 담화가 얼마나 붙어 있나
        - PMI 양수 : A가 말한 뒤에 B가 바로 받아서 말하더라
        - PMI 음수 : 둘 사이에 소통이 단절되어있다.
        
        | **구분** | **PMI 높음 (내용 끈끈함)** | **PMI 낮음 (내용 산만함)** |
        | --- | --- | --- |
        | **PPL 낮음**
        (흐름 매끄러움) | **🏆 최상의 회의 (Deep Flow)**
        주제도 깊고, 티키타카도 잘됨.
        *(이상적인 전략 회의)* | **🗣️ 잡담/친목 (Small Talk)**
        대화는 즐겁고 매끄럽지만,
        남는 내용은 없음. |
        | **PPL 높음**
        (흐름 끊김/낯섦) | **⚔️ 격렬한 논쟁 (Debate)**
        주제는 확실하지만,
        서로 반박하느라 흐름이 거침. | **💥 최악의 회의 (Chaos)**
        주제도 중구난방이고,
        서로 무슨 말 하는지 모름. |
