## 새로운 모델 탐색
🇰🇷 추천하는 한국어 NER 모델
1. Hugging Face Transformers (고성능 및 표준)
가장 성능이 뛰어나고 유연하게 사용할 수 있는 방법입니다. Python의 transformers 라이브러리와 함께 사용합니다. 특히 KLUE (Korean Language Understanding Evaluation) 벤치마크의 NER 데이터셋으로 학습된 모델들이 우수한 성능을 보입니다.
모델 추천 1: klue/roberta-large (KLUE-NER 파인튜닝 모델)
특징: 현재 한국어 NER에서 가장 높은 성능을 보이는 모델 중 하나입니다. KLUE 벤치마크에서 SOTA(최고 성능)를 기록했습니다.
적합한 경우: 정확도가 가장 중요한 서비스나 연구에 적합합니다.

모델 추천 2: monologg/koelectra-base-v3-discriminator (NER 파인튜닝 모델)
특징: roberta-large 모델보다 가벼우면서도 준수한 성능을 제공합니다. 속도와 정확도의 균형이 좋습니다.
사용 방법: Hugging Face의 transformers 라이브러리에서 pipeline 기능을 사용하면 몇 줄의 코드로 쉽게 NER을 수행할 수 있습니다.

2. Pororo (간편한 올인원 라이브러리)
카카오브레인에서 개발한 한국어 NLP 라이브러리입니다. 사용하기 매우 간편하게 패키징되어 있어, 복잡한 설정 없이 빠르게 NER 기능을 구현하고 싶을 때 유용합니다.
특징: Pororo(task="ner", lang="ko")와 같이 간단한 코드로 모델을 불러와 바로 사용할 수 있습니다.
내부 모델: Pororo 역시 내부적으로는 BERT 기반의 고성능 모델을 사용합니다.
적합한 경우: 빠르고 간단하게 NER 기능을 프로토타이핑하거나 구현해야 할 때 좋습니다.

## 학습용 데이터 탐색
1. 회의록 데이터: 「회의 음성」
'회의록' 텍스트와 그에 대한 NER 태그가 필요할 때 가장 적합한 데이터셋입니다. 음성 데이터와 함께 JSON 형식의 전사(transcript) 파일이 제공되며, 이 전사 파일 안에 개체명(NE) 태그가 포함되어 있습니다.
데이터셋 이름: 회의 음성
구체적인 주소: https://aihub.or.kr/aidata/30709
주요 특징:
다양한 주제의 회의(일반, 금융, 기술 등) 음성 및 전사 데이터
JSON 파일 내에 발화자, 발화 시간, 그리고 'NE'(Named Entity) 태그가 포함되어 있습니다.
'홍기님'과 같은 구어체 이름과 직책을 학습시키기에 적합합니다.

2. 일상 대화 데이터: 「한국인 대화음성」
'일상 대화' 문맥에서 사람, 장소 등을 찾는 모델을 학습시킬 때 가장 적합한 데이터셋입니다. 이 데이터셋은 설명서에 **'개체정보'**를 제공한다고 명시되어 있습니다.

일상적인 자유 대화(음성) 및 전사 데이터
JSON 파일 형태로 제공되며, 설명서에 따르면 "개체정보(NE), 형태소분석" 등의 후처리 가공이 되어 있다고 명시되어 있습니다.
'홍기님인 거 알아'와 같은 자연스러운 구어체 문맥을 학습시키기에 좋습니다.
