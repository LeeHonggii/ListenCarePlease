## 현재 상태

### 기술 스택
- **Frontend:** React 18 + Vite + Tailwind CSS
- **Backend:** FastAPI + SQLAlchemy + MySQL
- **AI 모델:** Whisper (STT), Senko/NeMo (Diarization), KoNER (이름 감지)

### 구현 완료 기능
✅ 파일 업로드 (m4a, wav 등)
✅ 음성 전처리 (VAD, 노이즈 제거)
✅ STT (Whisper Local/API)
✅ 화자 분리 (Senko/NeMo)
✅ 이름 감지 및 화자 매칭
✅ 대시보드 (통계, 최근 파일, 처리 현황)
✅ 결과 조회 및 수정
✅ 다크모드

### 추가 예정 기능
🎯 **RAG 시스템** (실제 구현)
📊 **더미 기능들** (UI만 구현, 실제 데이터 미연동)

## 기능 분류

### 실제 구현 기능 (백엔드 연동)
1. **RAG 시스템**
   - 회의록 벡터 DB 인덱싱 (ChromaDB + SentenceTransformer)
   - 사용자 질문에 대한 의미 기반 검색
   - LLM 기반 답변 생성 (출처 포함)
   - 관련 발화 타임스탬프 제공

### 더미 기능 (UI만 구현)
1. **회의 템플릿 선택**
   - 정보 교류용 / 문제 해결용 / 협업용
   - 템플릿별 아이콘 및 설명
   - 선택 시 결과 페이지 스타일 변경 (색상, 레이아웃)

2. **날짜 입력**
   - 업로드 페이지에 DatePicker 추가
   - 회의 날짜 메타데이터로 저장 (DB 연동은 나중에)
   - 결과 페이지 헤더에 날짜 표시

3. **정량적 지표 시각화** (차트 더미)
   - 발화량 도넛 차트 (화자별 %)
   - 발화 빈도 바 차트
   - 시간대별 발화 분포 라인 차트
   - 발화 속도 게이지 차트
   - TTR (어휘 다양성) 바 차트
   - 침묵 구간 히트맵
   - **데이터:** 랜덤 또는 하드코딩된 값 사용

4. **LLM 모델 선택**
   - 업로드 페이지에 라디오 버튼 추가
   - 선택지: None / 엑사원 / QWEN
   - UI만 존재, 실제 LLM 호출은 RAG 시스템만 사용

5. **요약 탭** (더미)
   - 결과 페이지에 "요약" 탭 추가
   - 하드코딩된 요약 텍스트 표시
   - 주요 결정사항, 액션 아이템 리스트

```jsx

  {/* Sidebar */}
  <Sidebar />

  {/* Main Content */}
  <div className="flex-1 flex flex-col">
    <TopBar />
    <main className="flex-1 overflow-y-auto">
      {children}
    </main>
  </div>
</div>
```

#### 컴포넌트
- `StatsCard`: 아이콘, 숫자, 레이블
- `ProgressRing`: 원형 진행률 (0-100%)
- `FileTable`: 정렬, 필터, 페이지네이션 지원

   - DatePicker
   - TemplateSelector

   - 사이드바 네비게이션
   - 탭 전환

   - Recharts 설치 및 기본 차트 구현
   - 하드코딩된 데이터로 렌더링
