# 파이프라인 구상
1) 문맥 기반 난해/중요 단어 추출 + 주석 파이프라인

전사 파일
   ↓
[LLM] 대화 전체 문맥 파악
   ↓
[LLM] 
- 대화 중에서 따로 설명된 단어들
- 사람들이 잘 이해 못한 것처럼 보이는 단어들
  → 후보 단어 리스트로 선택
   ↓
[LLM 또는 외부 사전] 
- 각 단어에 대한 뜻/정의/간단 설명 생성
   ↓
"단어 : 뜻" 형태의 주석 리스트 생성
   ↓
전사 파일 마지막에

----------------------------------
[용어 주석]
단어1 : 뜻1
단어2 : 뜻2
단어3 : 뜻3
...


2) NER 기반 전문 용어 추출 + 주석 파이프라인

전사 파일
   ↓
화자 라인 제거
(예: "참석자 1", "참석자 2" 등 제거)
   ↓
텍스트만 추출 (발화 내용만)
   ↓
[NER 모델/LLM] 
- 문장 단위 NER 태깅
   ↓
전문 용어라고 판단되는 엔티티만 필터링
(도메인/기술/제품/서비스명 등)
   ↓
set() 으로 중복 제거
→ 고유 전문 용어 리스트
   ↓
[LLM 또는 외부 사전] 
- 각 전문 용어의 뜻/정의/간단 설명 생성
   ↓
"단어 : 뜻" 형태의 주석 리스트 생성
   ↓
전사 파일 마지막에

----------------------------------
[전문 용어 주석]
용어1 : 뜻1
용어2 : 뜻2
용어3 : 뜻3
...
