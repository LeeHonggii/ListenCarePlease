# GLiNER
[GLiNER_github](https://github.com/urchade/GLiNER)

[taeminlee/gliner_ko](https://huggingface.co/taeminlee/gliner_ko)
[urchade/gliner_multi-v2.1](https://huggingface.co/urchade/gliner_multi-v2.1)

~~~
model = GLiNER.from_pretrained("taeminlee/gliner_ko")

text = """
저도 그래서 딱 생각한게 진짜 처음에 ABC로 나오다가 우리처럼 이제 막 상주님, 홍기님 그렇게 언급됐을 때부터 그 다음에 태깅되는 거 아니면.
"""
~~~
~~~
상주 => PERSON
홍기 => PERSON
태깅 => PERSON
~~~

~~~
text = "저도 그래서 딱 생각한게 진짜 처음에 ABC로 나오다가 우리처럼 이제 막 상주님, 홍기님 그렇게 언급됐을 때부터 그 다음에 태깅되는 거 아니면."

labels = ["person"]

entities = model.predict_entities(text, labels)
~~~
~~~
상주님 => person
홍기님 => person
~~~
